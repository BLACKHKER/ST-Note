## Linux命令

### 一、基本命令

#### 1.1 终端

| 操作    | 功能                                 |
| ------- | ------------------------------------ |
| su root | 切换root用户(密码隐藏，且不显示长度) |
| clear   | 清空当前控制台历史信息               |





#### 1.2 目录

| 操作                 | 功能                                                     |
| -------------------- | -------------------------------------------------------- |
| cd 相对路径/绝对路径 | 切换到指定目录地址                                       |
| cd ..                | 切换到当前目录的上一级                                   |
| cd ~/cd              | 切换到当前用户的主目录，/home/用户名(等同于cd什么都不写) |
| cd /                 | 切换到根目录                                             |
| pwd                  | 查看当前所在目录位置                                     |





#### 1.3 文件

> 蓝色:文件夹，黑色:文件，绿色:可执行文件，红色:压缩包

| 操作                         | 功能                                                         |
| ---------------------------- | ------------------------------------------------------------ |
| ls                           | 查看当前所在目录下所有文件，**不包含隐藏文件**               |
| ls -a                        | 查看当前所在目录下所有文件，**包含隐藏文件**                 |
| ls -l                        | 查看当前所在目录下所有文件及其详细信息(权限、创建人、创建时间) |
| ls -la                       | 查看当前所在目录下所有文件，包含隐藏文件，及其详细信息       |
| touch 文件名(包含后缀)       | 在当前目录创建指定文件名的文件                               |
| cp 文件名 新文件名           | 复制一个文件，重命名为新文件名                               |
| cp 文件名 文件夹名           | 复制文件到指定文件夹                                         |
| cp 文件名 文件夹名/新文件名  | 复制文件到指定文件夹并重命名为新文件名                       |
| rm 文件名(包括后缀)          | 删除文件                                                     |
| rm -f 文件名                 | 删除文件，没有提示Y/N                                        |
| rm *.后缀                    | 删除指定后缀的文件                                           |
| mv 文件名 文件夹名           | 将文件移动到指定文件夹                                       |
| mv 文件名 新文件名(包括后缀) | 重命名文件(可以修改后缀)                                     |
| cat 文件名(含后缀)           | 显示文件内容                                                 |





#### 1.4 文件夹

| 操作                                         | 功能                                                         |
| -------------------------------------------- | ------------------------------------------------------------ |
| mkdir 文件夹名                               | 在当前目录创建一个文件夹                                     |
| mkdir 文件夹1 文件夹2 文件夹3                | 在当前目录创建多个文件夹                                     |
| mkdir 文件夹名1/文件夹2/...                  | 在某父文件夹下创建子文件夹，支持多层<br />(父文件夹必须存在) |
| mkdir -p 文件夹名1/文件夹2/文件夹名3/文件夹4 | 在父文件夹下一次性创建多个子文件夹<br />(父文件夹可以不存在) |
| rm -r 文件夹名                               | 删除指定文件夹                                               |





#### 1.5 压缩文件/文件夹

| 操作                                   | 功能                             |
| -------------------------------------- | -------------------------------- |
| tar -xf 压缩文件名                     | 解压文件                         |
| tar -cf 压缩后文件名 被压缩文件/文件夹 | 压缩指定文件/文件夹为.tar文件    |
| tar -czf 被压缩文件/文件夹             | 压缩指定文件/文件夹为.tar.gz文件 |





#### 1.6 权限

> 权限名：r：read读，w：write写，x：execute执行
>
> 文件没有读权限，打开文件不显示文件内容，没有写权限，可以通过!强制写入

| 操作                          | 功能                   |
| ----------------------------- | ---------------------- |
| chmod                         | 权限名 文件/文件夹名   |
| chmod +/-权限名 文件/文件夹名 | 给文件/文件夹加/减权限 |
| chmod 三位二进制权限数 文件名 | 给文件/文件夹加/减权限 |

### 二、其他命令

#### 2.1 编辑网卡设置

```shell
cd /etc/sysconfig/network-scripts		-- 切换到网卡配置文件所在目录
vi ifcfg-ens33		-- 使用vi编辑器编辑网卡配置文件
esc		-- 进入打开状态
:wq		-- 退出
sudo service network restart	-- 重置/重启网络设置
```





#### 2.2 网络相关

```shell
ip addr										-- 查看IP信息
sudo yum install net-tools					-- 安装netstat包(才能使用sudo netstat -tlnp)
sudo netstat -tlnp							-- 查看进程占用的所有端口
systemctl stop firewalld					-- 先停止系统防火墙
systemctl disable firewalld					-- 并禁用自启动
```

### 三、VI编辑器

#### 3.1 工作状态

> 输入模式：i切换到INSERT编辑状态
> 退出:输入":"光标移到下面，输入以下指令：
> q:不保存退出
> wq:写入并退出

##### 3.1.1 正常状态(打开时状态)

> 打开文件时的状态，不能编辑文件，按i进入编辑状态，输入":"进入命令状态
>

| 操作              | 功能                                   |
| ----------------- | -------------------------------------- |
| yy(快速双击y)     | 复制当前光标所在行                     |
| nyy(2、3..)yy     | 从当前光标所在位置开始复制n行          |
| p                 | 粘贴刚才复制的行                       |
| dd                | 剪切当前行                             |
| u                 | 撤销                                   |
| ctrl + r          | 恢复上一次撤销的动作                   |
| / 关键字          | 查询关键字(高亮，按n切换下一个高亮)    |
| :s/str1/str2/     | 替换当前行第一个str1为str2             |
| :s/str1/str2/g    | 替换当前行中所有str1为str2             |
| :%s/str1/str2/g   | 替换所有行中的str1为str2               |
| :ms/str1/str2/    | 替换第m行的第一个str1为str2            |
| :m,ns/str1/str2/g | 替换第m行开始到第n行中所有的str1为str2 |



##### 3.1.2 编辑状态

> 可以对文件进行编辑，按esc切换成打开状态
>



##### 3.1.3 命令状态

用来输入一些指令对文件进行保存、退出、查询、替换等操作

| 操作   | 功能                               |
| ------ | ---------------------------------- |
| !      | 强制                               |
| w      | 写入、保存(等同ctrl+s)，write      |
| q      | 退出，quit                         |
| q!     | 退出，强制执行                     |
| wq     | 写入并退出                         |
| set nu | 显示行号(临时设置，关闭文件后消失) |





#### 3.2 设置永久行号

```shell
cd ~ cd 空格	-- 切换到用户目录
vi .vimrc	-- 创建并编辑文件vimrc
set nu	-- 在文件中设置行号
```

### 四、常用软件包安装

#### 4.1 Cmake

> 执行./bootstrap前需确保已安装OpenSSL
>
> ```shell
> sudo apt install libssl-dev
> ```

```shell
sudo apt-get install build-essential
wget https://github.com/Kitware/CMake/releases/download/v3.26.3/cmake-3.26.3.tar.gz
tar -zxvf cmake-3.26.3.tar.gz
cd cmake-3.26.3
./bootstrap
make -j8
sudo make install
```
